A PHP Framework for API and Web Development

git clone https://github.com/Syntax-Squad/AstroSpark.git   

Contributor Guidelines

  1. Avoid Code Duplication : Do not create new solution or functions for existing features. Avoid code redundancy
  2. Keep Code Clean : Keep your code clean and readable.
  3. Discus Before Change: Open an issue on issues tab before you start to develop a feature or made a change in code to avoid unexpected code merging conflicts.
  4. Add Comments Properly : Add useful comments on each changes you made, for help to understand and collaborate easily.
  5. Push Your Changes to a New Branch: Always create a new branch and push your changes to that and open a pull request.
    ( Don't try to push main branch & don't use --force pushes )
  6. Use Forked Repo (optional) : You can fork the repo from Syntax-Squad/AstroSpark and use it to contribute.
  7. Describe and Document Your Changes: Give a full description about your changes while submitting a pull request.
Notice: we are aiming to provide a better framework for developers without having large dependency tree. Please avoid using another composer or npm packages for a feature or functionality. You want to develop the functionality from scratch with in this project. (You can use CDN links and open-source stable 3rd-party APIs for this project)

Pre-Requisites for Contribute

  • Basic knowledge of HTML, CSS and JavaScript.
  • Good knowledge in PHP.
  • Deep understanding about OOPS concepts.
  • Basics about API and how its works.
  • Basics of Larval Framework.
Note: These pre-requisites are helps you to understand and efficiently contribute to the project.
But we are welcome all interested contributors without coding knowledge too. (For Maintain Documentation and Pages)

What You Can Do ?
